# ACID

原子性,一致性,隔离性,持久性

# RU(read uncommitted)

一个终端可以读取另一个终端未提交的事务.产生脏读.

# RC(READ COMMITTED)

一个事务可以读取另一个已提交的事务，多次读取会造成不一样的结果，此现象称为不可重复读问题，Oracle 和 SQL Server 的默认隔离级别。
